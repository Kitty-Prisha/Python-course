What is a Variable?

Variables are something whose values can be changed in the process.

 

What is a Procedure?

Procedures are like functions that are created to be used again and again. Consider you want a robot to fill the water in the bottles again and again. You will have to give it a command -” Hey Chitti Fill the water Bottle''. To ease it, you can create a function (for example Fill_Bottle) and use it whenever you want the robot to fill the bottle. To activate this function you can define the function as:

Whenever I clap, fill the Bottle.
